## Loop Contract

Each loop must complete:

1. Think: hypothesis, strongest counterexample, minimum experiment.
2. Explore: repo, legacy reference, external project/model/research scan.
3. Experiment: one minimum verifiable mainline correction.
4. Evaluate: deterministic checks, scripted real-entry, GPT-5.5 judge, human smoke when required.
5. Summarize: evidence path, failure taxonomy, what changed, what remains unproven.
6. Continue: choose the next safe loop or record a blocker.
