# Ego Joi-Inspired Companion Roadmap v1

## Summary

Build a Joi-inspired companion experience roadmap for `EgoOperator` by extracting interaction mechanisms from *Blade Runner 2049*'s Joi reference surface, not by copying Joi dialogue or persona text. The target mechanism is a warmer, more continuous, more emotionally attuned, and eventually embodied operator companion.

## Structure Risk Check

- Real target: translate a user-perceivable companion ideal into testable `EgoOperator` mechanisms.
- Drift risk: long dialogue capture or fixed persona templates would create prompt-copy behavior instead of durable interaction primitives.
- Contract first: research output must become interaction beats, eval samples, and task-board issues before runtime changes.
- Counterexample gate: if scripted/GPT-5.5-judged smoke passes but real CLI still feels detached, the roadmap has not proved user-perceived companionship.
- Current validation: roadmap and local scripted tooling only; no stable benefit, runtime efficacy, live autonomy, durable memory efficacy, independent awareness, or consciousness claim.
