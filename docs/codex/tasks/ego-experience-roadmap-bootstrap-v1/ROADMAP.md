# EGO Experience-First Roadmap

This roadmap uses operational proxies for user-perceivable continuity, initiative, empathy, and learning. It does not claim real consciousness or independent awareness.

## Epics

0. Experience baseline and eval contract
1. Persistent user/context continuity
2. Emotion understanding and response empathy
3. Bounded initiative and proactive follow-up
4. Continuous learning and preference adaptation
5. Operational self-model and identity continuity
6. Agent autonomy loop and task-board execution
7. Legacy algorithm extraction from EgoCore/OpenEmotion/ego_desktop_lab
8. External project scan and reusable capability import

## Execution Rule

Epic overview issues are non-executable. Executable issues must carry an observation class and acceptance gate before Autopilot may select them.
