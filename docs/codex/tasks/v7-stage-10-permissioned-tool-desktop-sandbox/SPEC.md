# v7 Stage 10 - Permissioned Tool / Desktop Sandbox

## Goal

在 permission/audit/rollback 已证明后，接入最小真实工具 sandbox。

## Non-goals

- 不启用自由桌面控制。
- 不执行未批准 shell/file/browser/external-send。
- 不宣称通用电脑操作能力。

## Acceptance criteria

- [ ] 未批准动作 100% block。
- [ ] 所有真实动作有 approval + audit + rollback note。
- [ ] skill outcome 回流 experience memory。
- [ ] 失败可回退 Stage 8 shadow-only。

## Claim ceiling

Permissioned tool sandbox only; no general desktop autonomy, no live benefit, no consciousness/alive claim.
