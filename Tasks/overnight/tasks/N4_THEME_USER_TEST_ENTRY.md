# N4 主题 — 用户可测入口与诊断主题

## 任务类型
执行入口设计 / 诊断壳 / 使用说明主题

## 目标
让用户第二天不用读大量代码，就能直接测试 Proto-Self Kernel v1 当前效果。

## 当前层级
用户可体验收口层

## 本主题允许连续推进的子任务
- N4A 用户测试合同冻结
- N4B 只读诊断入口或等价脚本
- N4C 固定测试场景包
- N4D 操作手册、对照表与主题报告

## 主题级成功判据
1. 有用户可直接执行的测试说明
2. 有固定测试场景
3. 有只读诊断入口或等价脚本
4. 有“预期现象 / 异常现象 / 排查入口”对照表
5. 主题报告存在

## 主题级硬约束
- 不要把调试壳设计成能越权改状态
- 不要把主观感受当唯一验收
- 不要要求用户读很多内部文件才能测试
