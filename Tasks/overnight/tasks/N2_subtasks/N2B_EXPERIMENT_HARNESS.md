# N2B — 实验脚手架与 artifact 通道

## 目标
搭出本主题的最小实验执行壳，不要求华丽，只要求：
- 可跑
- 可记录
- 可比较
- 可回读

## 必做项
1. 统一实验入口
2. 统一 artifact 输出目录
3. 统一结果摘要格式
4. 能读取关键 trace 字段
5. 能对比至少两组实验结果

## 成功判据
- 可运行的实验入口存在
- 至少能生成一个 artifact
- 至少能输出一份结构化结果摘要

## 硬约束
- 不要为实验引入第二套本体状态
- 不要让 harness 变成新的黑箱真相源

## 失败出口
若脚手架无法在可信环境中运行：
- 不进入 N2C
- 报 `blocked`
