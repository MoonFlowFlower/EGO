# N3 主题 — 泛化、误聚合与反证主题

## 任务类型
泛化验证主题批次

## 目标
确认当前 Proto-Self Kernel v1 的有效性不是只建立在少数样本上，并暴露 coarse intent bucket 的误聚合边界。

## 当前层级
泛化验证层

## 本主题允许连续推进的子任务
- N3A 样本合同与观测指标冻结
- N3B 应聚合样本与命中检查
- N3C 应区分样本与误聚合检查
- N3D Replay 一致性与风险清单

## 主题级成功判据
1. 至少有一组应聚合样本
2. 至少有一组应区分样本
3. 至少完成一组 replay / trace 一致性检查
4. 输出主题报告，明确：
   - 已证实范围
   - 未证实范围
   - 风险与反例

## 主题级硬约束
- 不允许为通过而把样本写得太像
- 不允许忽略失败样本
- 不允许把“暂未发现问题”写成“问题不存在”
