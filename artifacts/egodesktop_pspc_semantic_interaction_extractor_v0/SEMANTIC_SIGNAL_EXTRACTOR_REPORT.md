## Implemented Behavior

PSPC Reply Preview no longer uses raw-text keyword classification as its main category route. It consumes validated semantic event packets and applies deterministic proxy-state deltas to local window-lifetime preview state.

The local extractor helper emits strict JSON packets for interaction events such as `gift_or_care_offer`, `gentle_touch`, `affinity_statement`, `trust_probe`, `boundary_pressure`, and `fatigue_or_late_night`. If the extractor is unavailable, times out, or emits invalid JSON, the preview state is not updated.

EgoDesktop now schedules the semantic extractor after the backend chat turn and updates the debug overlay asynchronously. The current turn uses the previously confirmed PSPC state; the current user input can influence future preview context after extraction succeeds.

## Failure Meaning

If the overlay remains neutral during natural companion interaction, check `extractor_status` first. `extractor_unavailable` means the semantic helper did not return a valid packet; it is not evidence that the interaction lacked signal. If events are present but bars remain flat, the deterministic state-delta mapping is the likely defect.
